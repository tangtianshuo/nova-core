# Nova Web Demo

一个基于 React 的 Web 演示应用，展示 Nova Auth SDK 的认证功能。

## 功能

- React 18 + TypeScript
- Vite 快速开发和优化构建
- 现代响应式 UI，深色主题
- 实时认证状态管理
- 令牌验证和用户信息展示
- 友好的错误处理

## 前置条件

运行此演示前，请确保：

1. **Nova Core 后端** 运行在 `http://localhost:3000`
   - 配置并启动数据库
   - 确保 Prisma 迁移已执行

2. **Node.js** (v18 或更高版本) 已安装

## 快速开始

1. **安装依赖**:
   ```bash
   npm install
   ```

2. **启动开发服务器**:
   ```bash
   npm run dev
   ```

3. **打开浏览器**:
   访问 `http://localhost:5173`

## 环境变量

在 `web-demo` 目录创建 `.env` 文件（可选）：

```bash
# API URL (默认: http://localhost:3000)
VITE_API_URL=http://localhost:3000
```

## 使用说明

### 登录

1. 输入用户名或邮箱
2. 输入密码
3. 点击"登录"

### 查看用户信息

登录成功后，您将看到：
- 用户 ID 和用户名
- 访问令牌（部分遮蔽以保护安全）
- 令牌验证和退出登录按钮

### 验证令牌

点击"验证令牌"检查当前认证状态。

### 退出登录

点击"退出登录"清除会话并返回登录页面。

## 故障排除

### "网络错误"或"连接失败"

- 确保 Nova Core 后端运行在端口 3000
- 检查后端 URL 是否正确

### "无效凭证"

- 验证您使用的用户名/邮箱和密码是否正确
- 确保用户存在于数据库中

### 令牌验证失败

- 令牌可能已过期（默认：1 小时）
- 尝试重新登录获取新令牌

## 技术栈

- **React 18**: UI 框架
- **TypeScript**: 类型安全
- **Vite**: 构建工具和开发服务器
- **Nova Auth SDK**: 认证 SDK
